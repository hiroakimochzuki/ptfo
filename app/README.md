長期インターンシップの自主学習として作成したウェブアプリ。
ChatGPT_API_Blazor_1ではAZUREのapp serviceに接続し、ウェブアプリとして稼働するように作成したため、ローカルでは稼働しない。
ChatGPT_API_Blazor_2は、ローカルで稼働する。

ピザの注文アプリとチャットボットアプリを作成。
アプリ度動作画面についてはpngのフォルダに画像を入れている。

注文アプリ：ピザを注文するアプリ。ピザを選択し、決済情報として名前や住所を入力することで注文が確定する。確定した注文は注文番号から参照することができる。

チャットボット：chatGPTのような対話型のチャットボットアプリ。現在はAPIが接続されていないため、ローカルでも対話を行うことはできない。
